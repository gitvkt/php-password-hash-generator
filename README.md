# 🔐 PHP Password Hash Generator

Ferramenta web leve, responsiva e segura desenvolvida em PHP para geração de hashes de senhas utilizando as funções nativas `password_hash()` (Bcrypt / Argon2id).

## 🚀 Funcionalidades
- Interface moderna em **Tailwind CSS** com suporte a modo escuro.
- Seleção de algoritmos nativos (BCRYPT / Argon2id).
- Ajuste de fator de custo (*cost factor*).
- Medidor de tempo de execução da geração do hash.
- Botões de cópia rápida para o Hash e para comandos SQL de atualização (`UPDATE`).

## 🛠️ Requisitos
- PHP 7.4 ou superior.
- Servidor Web (Apache, Nginx, cPanel, etc.).
